# ecommerce-sql-project
An end-to-end e-commerce SQL project analyzing customer behavior, revenue trends, and product performance using joins, aggregations, and business-driven queries.

# E-commerce Sales Analysis (SQL Project)

## Project Overview
This project analyzes an e-commerce dataset using SQL to extract business insights such as customer behavior, sales trends, and product performance.

## Tools Used
- SQL (MySQL Workbench)

## Key Analysis
- Top 10 customers by revenue
- Monthly sales trend
- Repeat vs one-time customers
- Product-wise sales and profit

## Database Structure
- customers
- products
- orders
- payments

## Key SQL Concepts Used
- JOIN
- GROUP BY
- ORDER BY
- Aggregate Functions (SUM, COUNT, AVG)

## How to Run
1. Run `create_tables.sql`
2. Run `insert_data.sql`
3. Run `analysis_queries.sql`

## Outcome
Identified high-value customers, sales trends, and top-performing products for better business decision-making.
